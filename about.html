About MediPredict AI

Clinical triage simulation parameters, technologies, and team.

Core Vision

MediPredict AI is designed as a clinical triage simulator. The platform models how modern, client-side, zero-database architectures can run advanced diagnostic classifications with zero network overhead. By running models locally in client browsers, the platform eliminates server function round-trip times entirely.

🧬 Naive Bayes Mathematics

Instead of hosting high-latency, resource-heavy Python Flask models, MediPredict AI computes predictions using a custom **Naive Bayes Classifier** in pure client-side JavaScript:

ln P(D | S) = ln P(D) + ∑ ln P(s_i | D) + ∑ ln(1 - P(s_j | D))

By calculating conditional probability metrics in logarithmic space (log-likelihoods), the classifier prevents mathematical underflows. It applies Laplace smoothing and normalizes the results into relative percentages using a Softmax function before displaying matches.
